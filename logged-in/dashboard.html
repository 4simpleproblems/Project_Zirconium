Navigation Test Notes

The horizontal scrolling menu should be visible above this content.

All menu items should be clickable.

When logged in, click your profile icon to see your account menu.

Hover over the three grey buttons next to your profile picture to pin/unpin pages.

Since "dashboard.html" is not defined in `page-identification.json`, the pin buttons should not automatically highlight an active tab. You can, however, modify `navigation.js` to add this page ID if desired.
