4SP Documentation

Your comprehensive guide to getting the most out of the Essential Student Toolkit.

1. Getting Started

Account Authentication

To access the full suite of tools, you must first log into or create your 4SP account. Use the 'Get Started Instantly' button on the homepage or navigate directly to the **Authentication** page.

Email Verification

After registration, you will be prompted to verify your email address to complete your registration. You can check the verification status from the **Verify Email** page.

2. Navigation Features

The Pin Button

The Pin Button is a **persistent Quality-of-Life feature** located next to the user authentication icon in the navigation bar, offering quick access to your most-used page.

  • **Pinning & Repinning:** Click the Pin Button on any page to pin that page. The Repin option appears in the right-click menu if the current page is different from the one currently pinned, or if no page is pinned.
  • **Options Menu:** You can **right-click** on the Pin Button to access its context menu.
  • **Visibility:** The button can be **hidden** via the right-click menu. If hidden, a **Show Pin Button** option appears in your main user dropdown menu.

Right-click for options!

3. Core Applications

Dashboard

The authenticated landing page for users, providing a clean interface that displays the current date and time.

Messenger V2

A secure, **peer-to-peer messenger** that supports **group chat** functionality, **image sharing**, and utilizes a **hybrid message storage** system.

Interactive Dictionary

A utility for students, offering a simple interface for searching and displaying word definitions powered by `dictionaryapi.dev`.

Notes

A simple note-taking application featuring **password-encrypted storage** for your private notes.

Settings

The page for managing your user account and configuring application options. This includes the ability to customize features like the **navigation bar color**.
