4SP Documentation

Your comprehensive guide to getting the most out of the **Essential Student Toolkit**.

1. Getting Started

Account Authentication

Access the full suite of tools by logging into or creating your 4SP account. The **Authentication** page supports both standard **email/password** registration and social sign-in methods like **Google**, **GitHub**, and **Microsoft**.

Email Verification

Only **email/password accounts** must verify their email address to complete registration. Accounts created using **social sign-in methods** (Google, GitHub, Microsoft) are typically **pre-verified** and do not require this step.

2. Navigation Features

The Pin Button

The Pin Button is a **persistent Quality-of-Life feature** located in the navigation bar, offering quick, one-click access to your **most-used page**.

  • **Pinning & Repinning:** Click the Pin Button on any page to pin that page. The **Repin option** is available via **right-click** if the current page is different from the pinned page, or if no page is currently pinned.
  • **Options Menu:** You can **right-click** on the Pin Button to access its context menu.
  • **Visibility:** The button can be **hidden** via the right-click menu. If hidden, a **Show Pin Button** option appears in your main user dropdown menu.
  • **Pin Hint:** A **one-time hint** appears on the first click of the Pin Button to explain its primary function.

3. Core Applications

Dashboard

The primary landing page for **authenticated users**. It provides a clean, distraction-free interface that prominently displays the **current date and time**. This allows you to quickly verify your access and get a clear sense of the time without navigating away from the main application environment.

  • **Default Landing:** The page users are redirected to upon **successful sign-in**.
  • **Time Display:** Features a real-time, high-contrast **digital clock** display.

DailyPhoto

A social feature that encourages mindful sharing by limiting users to **three photos per day** to share with their friends. This creates a focused, high-quality sharing experience.

  • **Friend System:** Users can send and manage **friend requests** to create a private sharing circle.
  • **Daily Limit:** Users can only upload a maximum of **three photos** within a 24-hour period.
  • **Profile Display:** Displays shared photos from your friends in a clean, scrollable feed.

Interactive Dictionary

A quick-reference utility for students, offering a simple interface for **searching and displaying word definitions**. It leverages the comprehensive **dictionaryapi.dev** service.

  • **Phonetics:** Provides phonetic spellings and, where available, **audio pronunciations**.
  • **Synonyms & Examples:** Displays **synonyms**, **antonyms**, and **example** sentences for improved comprehension.
  • **Part-of-Speech Grouping:** Definitions are clearly organized by **Part of Speech** (e.g., Noun, Verb, Adjective).

Notes

A secure, simple note-taking application designed for privacy, featuring **password-encrypted storage** for your private notes.

  • **Client-Side Encryption:** Uses the **Crypto-JS** library for secure encryption, meaning only you can read your notes.
  • **Lock/Unlock:** Notes are locked behind a user-defined **password**.
  • **Autosave:** Includes an **autosave** feature for a seamless note-taking experience.

Settings

The central hub for managing your user account and configuring application options. **Personalization features** are managed here.

  • **Navigation Color:** Customize your experience by selecting, **saving, and restoring your desired navigation bar color**.
  • **Restore Defaults:** Option to **restore** all saved settings to their default values.
  • **Account Management:** Links for managing your **password**, **email**, and **social** account connections.

Humanity Agent (AI)

A **context-aware AI persona** designed to assist with your educational needs.

  • **Activation:** Quickly activate the agent using the **Ctrl + \** keyboard shortcut.
  • **Dynamic Persona:** Features an **intelligent system instruction** that automatically changes based on the nature of your query.
  • **Web Search:** Includes a **Web Search** toggle in the settings menu, allowing the agent to access **up-to-date information**.
  • **KaTeX Support:** Renders mathematical formulas and equations using **KaTeX** for **high-quality display**.
